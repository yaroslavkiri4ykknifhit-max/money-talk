# 🚀 MONEY-TALK Simple - Полная инструкция

## 📦 Что это?

Простая версия Telegram Mini App для обучения продажам:
- ✅ Один HTML файл (можно залить на GitHub Pages)
- ✅ Загрузка контента из Google Sheets
- ✅ Простой Python бот
- ✅ Без сложного backend

---

## 📋 Шаг 1: Настройка Google Sheets

### 1.1 Создайте таблицу

1. Откройте [Google Sheets](https://sheets.google.com)
2. Создайте новую таблицу
3. Назовите её: **MONEY-TALK Content**

### 1.2 Создайте вкладку "Modules"

Создайте первую вкладку с названием **Modules** и заполните:

| id | title                    | order |
|----|--------------------------|-------|
| 1  | Введение в продажи       | 1     |
| 2  | Построение отношений     | 2     |
| 3  | Техники закрытия сделок  | 3     |

### 1.3 Создайте вкладку "Lessons"

Создайте вторую вкладку с названием **Lessons** и заполните:

| id | module_id | title                  | content                                                    | order |
|----|-----------|------------------------|------------------------------------------------------------|-------|
| 1  | 1         | Добро пожаловать       | # Добро пожаловать в MONEY-TALK\n\nЗдесь вы научитесь... | 1     |
| 2  | 1         | Основы продаж          | # Основы продаж\n\n**Ключевые принципы:**\n- Слушайте... | 2     |
| 3  | 1         | Психология клиента     | # Психология клиента\n\nПонимание мотивации...            | 3     |
| 4  | 2         | Первый контакт         | # Первый контакт\n\nКак правильно начать...               | 1     |
| 5  | 2         | Построение доверия     | # Построение доверия\n\nДоверие - основа...               | 2     |

**Важно:** В колонке `content` можно использовать простой Markdown:
- `# Заголовок` - большой заголовок
- `## Подзаголовок` - средний заголовок
- `**жирный текст**` - жирный
- `*курсив*` - курсив
- `\n` - перенос строки

### 1.4 Сделайте таблицу публичной

1. Нажмите **Файл** → **Опубликовать в интернете**
2. Выберите **Вся таблица**
3. Нажмите **Опубликовать**
4. Скопируйте **ID таблицы** из URL

Пример URL:
```
https://docs.google.com/spreadsheets/d/1ABC123xyz456/edit
                                      ↑
                                   Это ID
```

## 📋 Шаг 2: Настройка Telegram бота

### 2.1 Создайте бота

1. Откройте Telegram
2. Найдите [@BotFather](https://t.me/BotFather)
3. Отправьте команду: `/newbot`
4. Введите имя: `MONEY-TALK`
5. Введите username: `moneytalk_bot` (должен заканчиваться на `_bot`)
6. **Скопируйте токен** (выглядит как `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`)

### 2.2 Настройте Mini App

1. Отправьте @BotFather команду: `/newapp`
2. Выберите вашего бота
3. Введите название: `MONEY-TALK`
4. Введите описание: `Обучение продажам`
5. Загрузите фото (640x360 пикселей)
6. Отправьте демо GIF (опционально)
7. **Пока оставьте URL пустым** (добавим после GitHub Pages)

## 📋 Шаг 3: Загрузка на GitHub Pages

### 3.1 Создайте репозиторий

1. Откройте [GitHub](https://github.com)
2. Нажмите **New repository**
3. Название: `money-talk`
4. Сделайте **Public**
5. Нажмите **Create repository**

### 3.2 Настройте index.html

1. Откройте файл `index.html`
2. Найдите строку:
   ```javascript
   SHEET_ID: 'YOUR_GOOGLE_SHEET_ID',
   ```
3. Замените `YOUR_GOOGLE_SHEET_ID` на ID вашей таблицы

### 3.3 Загрузите файл

**Вариант A: Через веб-интерфейс**
1. В репозитории нажмите **Add file** → **Upload files**
2. Загрузите `index.html`
3. Нажмите **Commit changes**

**Вариант B: Через git**
```bash
cd ~/money-talk-simple
git init
git add index.html
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ВАШЕ_ИМЯ/money-talk.git
git push -u origin main
```

### 3.4 Включите GitHub Pages

1. В репозитории откройте **Settings**
2. Слева выберите **Pages**
3. В **Source** выберите **main** branch
4. Нажмите **Save**
5. Подождите 1-2 минуты
6. **Скопируйте URL** (например: `https://yourusername.github.io/money-talk/`)

### 3.5 Добавьте URL в бота

1. Вернитесь к @BotFather
2. Отправьте: `/myapps`
3. Выберите ваше приложение
4. Нажмите **Edit Web App URL**
5. Вставьте URL вашего GitHub Pages
6. Готово!

## 📋 Шаг 4: Запуск Python бота

### 4.1 Установите Python

Убедитесь, что у вас установлен Python 3.8+:
```bash
python3 --version
```

### 4.2 Установите зависимости

```bash
cd ~/money-talk-simple
pip3 install -r requirements.txt
```

### 4.3 Настройте config.py

Откройте `config.py` и заполните:

```python
# Токен от @BotFather
BOT_TOKEN = "123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11"

# URL вашего GitHub Pages
WEB_APP_URL = "https://yourusername.github.io/money-talk/"

# ID Google Sheets (для справки)
GOOGLE_SHEET_ID = "1ABC123xyz456"
```

### 4.4 Запустите бота

```bash
python3 bot.py
```

Вы должны увидеть:
```
✅ Бот MONEY-TALK запущен успешно!
📱 Web App URL: https://yourusername.github.io/money-talk/
🛑 Нажмите Ctrl+C для остановки
```

## 🔧 Частые проблемы

### Проблема: "Ошибка загрузки данных"

**Решение:**
1. Проверьте, что таблица опубликована в интернете
2. Проверьте правильность ID таблицы в `index.html`
3. Проверьте названия вкладок: `Modules` и `Lessons` (с большой буквы)

### Проблема: Бот не запускается

**Решение:**
1. Проверьте токен в `config.py`
2. Убедитесь, что установлен `python-telegram-bot`:
   ```bash
   pip3 install python-telegram-bot==20.7
   ```

### Проблема: Mini App не открывается

**Решение:**
1. Проверьте URL в `config.py`
2. Убедитесь, что GitHub Pages активирован
3. Проверьте, что URL добавлен в @BotFather

## 📝 Обновление контента

Чтобы обновить контент:

1. Откройте вашу Google Таблицу
2. Измените данные в вкладках `Modules` или `Lessons`
3. Сохраните
4. Обновите страницу в Telegram (закройте и откройте заново)

**Изменения появятся сразу!** Не нужно ничего деплоить.
