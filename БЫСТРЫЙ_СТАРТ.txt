                    БЫСТРЫЙ СТАРТ (3 ШАГА)
═══════════════════════════════════════════════════════════════

ШАГ 1: Настройте Google Sheets
────────────────────────────────────────────────────────────────
1. Создайте Google Таблицу
2. Добавьте вкладки "Modules" и "Lessons"
3. Заполните данными (см. ИНСТРУКЦИЯ.md)
4. Опубликуйте в интернете
5. Скопируйте ID таблицы

ШАГ 2: Загрузите на GitHub Pages
────────────────────────────────────────────────────────────────
1. Откройте index.html
2. Замените YOUR_GOOGLE_SHEET_ID на ваш ID
3. Создайте репозиторий на GitHub
4. Загрузите index.html
5. Включите GitHub Pages
6. Скопируйте URL

ШАГ 3: Запустите бота
────────────────────────────────────────────────────────────────
1. Создайте бота у @BotFather
2. Откройте config.py
3. Вставьте BOT_TOKEN и WEB_APP_URL
4. Запустите:
   
   pip3 install -r requirements.txt
   python3 bot.py
