# 💰 MONEY-TALK Simple

Простая версия Telegram Mini App для обучения продажам.

## 🎯 Что это?

- ✅ Один HTML файл (можно залить на GitHub Pages)
- ✅ Загрузка контента из Google Sheets
- ✅ Простой Python бот для Telegram
- ✅ Без backend, без базы данных

## 🚀 Быстрый старт

### 1. Настройте Google Sheets

1. Создайте Google Таблицу
2. Сделайте её публичной (Файл → Опубликовать в интернете)
3. Скопируйте ID таблицы из URL

### 2. Настройте Telegram бота

```bash
pip install python-telegram-bot
python bot.py
```

### 3. Залейте на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите `index.html`
3. Включите GitHub Pages в настройках
4. Готово!

## 📁 Структура

```
money-talk-simple/

├── bot.py          # Telegram бот
├── config.py       # Настройки

```

## 🔧 Технологии
